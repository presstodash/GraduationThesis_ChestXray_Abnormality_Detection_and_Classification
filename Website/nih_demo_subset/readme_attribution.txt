Manually chosen subset of images from the publicly available NIH ChestXray dataset.
For demonstration purposes only!
Images in the NIH dataset are in PA / AP projections. The model was only trained on PA projection pictures. This could cause some unexpected behaviour.

Contains predominantly these featured classes:
Cardiomegaly
Pleural effusion
Nodule/Mass
No Finding
